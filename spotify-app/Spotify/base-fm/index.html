Setup Guide

  1. 1. Add this app to your favourites
  2. 2. Manually create a new playlist (or choose an old one)
  3. 3. Drag the playlist to this app's link in the left menu
  4. 4. Wait for tracks to be added to the playlist
  5. 5. Start the playlist playing
  6. 5. Keep this app open (you can moderate tracks on the playlist but always come back here)

If something breaks: Ctrl-Alt-R
